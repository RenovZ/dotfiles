# set asdf auto-completion

```fish
# intel chip
ln -s /usr/local/share/asdf/fish/vendor_completions.d/asdf.fish completions/

# apple silicon chip
ln -s /opt/homebrew/share/asdf/fish/vendor_completions.d/asdf.fish completions/
```
